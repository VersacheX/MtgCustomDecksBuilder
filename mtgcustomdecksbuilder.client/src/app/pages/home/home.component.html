MTG Deck Builder

System

  • OAuth2 Authentication using JWT Tokens
  • JWT Interception
  • Password Encryption
  • User Security Privileges
  • Component Base Class Inheritance
  • Dependency Injection
  • Entity Framework
  • Microservices
  • User Creation
  • E-mail integration. Password reset, account creation validation

Features and Integrations

  • Commander Deck Builder (Other formats require custom development)
  • Crawl EDHRec for sorted card suggestions
  • Crawl Commander Spellbook for both existing combos and suggested combos
  • Scryfall Integration for updated card data
  • Custom Game Rule Development based on existing formats (may be scrapped)

Frameworks

  • Built in .Net 5 / Angular 8
  • Upgraded to .Net 6 / Angular 12
  • Upgraded to .Net 8 / Angular 18

Appreciation to Scryfall, Edhrec and Commander Spellbook for their phenomenal services.
