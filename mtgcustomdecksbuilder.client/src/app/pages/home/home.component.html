MTG Deck Builder

This is the home page, the features in this application are designed as a template for future development. Some of the features included are:

  • OAuth2 Authentication using JWT Tokens
  • User Creation and Password encryption
  • E-mail integration. Password reset, account creation validation
  • Database connectivity
  • User Security Privileges
  • Custom component classes and inheritance
  • Dependency injection
  • Entity Framework
  • Microservices
  • Responsive Navigation system
  • Commander Deck Builder
  • Crawl EDHRec for sorted card suggestions
  • Crawl Commander Spellbook for both existing combos and suggested combos
  • Scryfall Integration for updated card data
  • Custom Game Rule Development based onexisting formats (may be scrapped)
